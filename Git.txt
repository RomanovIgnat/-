git rebase и git merge по сути решают одну и ту же задачу слияния веток, но merge буквально сливает ветки,
а rebase прямо переносит коммиты в конец целевой ветки.

merge

-A -- B -- C       -A -- B -- C
  \            =>    \         \
   D -- E             D -- E -- M-
   
rebase

-A -- B -- C        
  \           =>   -A -- B -- C -- D' -- E'           
   D -- E
